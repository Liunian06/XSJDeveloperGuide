# 贡献指南

感谢您对 Small Phone 的关注！我们欢迎任何形式的贡献，包括代码、文档、设计以及 Bug 报告。

## 如何开始

1.  Fork 本仓库
2.  创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3.  提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4.  推送到分支 (`git push origin feature/AmazingFeature`)
5.  开启一个 Pull Request

## 代码规范

请遵循现有的代码风格，并确保通过所有测试。

## 行为准则

请参阅我们的 [行为准则](/code-of-conduct)。
