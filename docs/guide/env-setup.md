# 环境搭建

本指南将帮助您搭建 Small Phone 的开发环境，涵盖 C++ 核心、Android (Jetpack Compose) 和 iOS (SwiftUI) 的开发需求。

## 前置要求

*   **CMake**: 3.20 或更高版本
*   **Android Studio**: 用于 Android 开发
*   **Xcode**: 用于 iOS 开发 (仅限 macOS)
*   **VS Code**: 推荐用于编写 C++ 核心代码
*   **Node.js**: v18.0.0+ (用于文档和脚本)

## 步骤 1: 克隆仓库

```bash
git clone https://github.com/your-org/small-phone.git
cd small-phone
```

## 步骤 2: 安装依赖

请根据您的目标平台，参考后续的详细安装指南。
