# 行为准则

## 我们的承诺

为了营造一个开放和热情的环境，作为贡献者和维护者，我们承诺让每个人都能在这个项目中和社区中获得无骚扰的体验，无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性取向如何。

## 我们的标准

有助于营造积极环境的行为包括：

*   使用欢迎和包容的语言
*   尊重不同的观点和经验
*   优雅地接受建设性的批评
*   关注什么对社区最有利
*   对其他社区成员表现出同情心

## 执行

如果发生违反本行为准则的情况，项目维护者有权并有责任采取任何他们认为适当的行动，包括但不限于删除评论、发出警告或禁止贡献者。
